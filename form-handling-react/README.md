 Form Handling in React

  Project Overview
This project demonstrates how to manage form data in React using two approaches:

1. Controlled Components* with `useState`
2. Formik + Yup** for advanced form handling and validation

The application implements a *User Registration Form* and simulates user registration using a *mock API*.

Objectives
- Understand how controlled components work in React
- Learn to manage form state and validation manually
- Transition to Formik for scalable and cleaner form handling
- Practice form validation using Yup
- Work with a mock API endpoint

Technologies Used
- React (Vite)
- JavaScript (ES6+)
- Formik
- Yup
- CSS
