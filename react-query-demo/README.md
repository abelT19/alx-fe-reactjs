Overview
This project demonstrates how to fetch, cache, and manage server data efficiently in a React application using **React Query**.  
Instead of manually handling loading states, errors, and caching, React Query simplifies data fetching and improves application performance and user experience.

The app fetches posts from the **JSONPlaceholder API** and shows how React Query handles caching and data updates automatically.

---

Project Objectives
- Learn how to integrate React Query into a React project
- Fetch data from a public API efficiently
- Handle loading and error states gracefully
- Understand caching and background refetching
- Improve UI responsiveness with minimal code

Technologies Used
- React (Vite)
- React Query
- JavaScript (ES6+)
- JSONPlaceholder API
