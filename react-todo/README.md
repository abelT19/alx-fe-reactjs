React Todo List with Testing

This project is a simple yet fully functional **Todo List application** built with React.  
It allows users to add, toggle, and delete todo items, while ensuring reliability through **unit tests written with Jest and React Testing Library**.

The goal of this project is to practice **component development** and **testing real user interactions** in a React application.

What This Project Covers
- Creating reusable React components
- Managing state with `useState`
- Handling user interactions (add, toggle, delete)
- Writing unit tests with Jest
- Testing UI behavior using React Testing Library

Technologies Used
- React (Vite)
- JavaScript (ES6+)
- Jest
- React Testing Library
